WebSocket 网关功能展示 — 在线状态 / 消息推送 / 弹幕直播 / 聊天室 / 订单追踪
在线状态
多用户独立窗口连接 WebSocket，实时查看在线状态，支持单播与广播消息
消息推送
按用户 / 房间 / 广播三种模式推送消息，WebSocket 实时回显
直播弹幕
接入真实 HLS 直播流，多用户发送弹幕，支持禁言管理与点赞
聊天室
多用户卡片式聊天窗口，支持单聊与群聊，实时消息同步
订单追踪
创建订单并通过 WebSocket 实时追踪状态变更，可视化时间线
管理后台
查看活跃连接、房间列表，支持踢出连接与广播管理
实时指标
实时监控连接数、用户数、成功率等关键指标，折线图可视化
